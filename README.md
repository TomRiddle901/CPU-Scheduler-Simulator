# CPU Scheduling Simulator

## Descrizione
Questo progetto consiste nella realizzazione di una **applicazione web** sviluppata con **HTML, CSS, JavaScript**
che permette di simulare il funzionamento di un algoritmo di **CPU Scheduling**

Questa applicazione consente di:
- Inserire i dati in alcuni processi (ID, Tempo di arrivo, Tempo di burst, priorità);
- Visualizzare i processi inseriti in una tabella;
- Osservare l'evoluzione dei processi tramite il **Diagramma di Gantt**;
- Calcolo dei principali parametri di scheduling:
    - Tempo di attesa medio;
    - Tempo di turnaround medio;
    - Tempo di risposta medio;
    - Efficenza del processore.

Questo progetto è utilizzato per lo studio degli algoritmi di sheduling dei sistemi operativi.

---

## Algoritmo di scheduling utilizzato
L'algoritmo che viene utilizzato è l'**FCFS (First Come First Served)**
